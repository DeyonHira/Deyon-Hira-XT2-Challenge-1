Wat wil ik bereiken met dit dashboard?

Ondanks ik XT2 een moeilijk vak vond (simpelweg omdat ik slecht ben in coderen), vind ik toch dat dit dashboard voor mijn doen goed gelukt is.
Ik heb vier soorten charts gemaakt en hoop hiermee de mensen/medewerkers van Space X, voldoende te informeren over de Space gegevens.


Waarom deze visualisatie?

Dit waren de visualisaties die mij op charts.js het meest aanspraken. Ook waren deze goed te combineren met mijn informatie.


Techniek

Ik heb voor de techniek de code van charts.js gebruikt. Dit was in het begin erg puzzelen. 
Maar na veel gelezen te hebben, werd het eigenlijk wel makkelijk om ze toe te passen. 
Ook ben ik er achter gekomen hoe de lettertype binnen de charts te veranderen.
Voor de pagina heb ik Flex Grids gebruikt, omdat ik dit ook al van de vorige leraar had gekregen (Jeffrey).
